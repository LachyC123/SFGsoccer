# Rift Rivals

Rift Rivals is a polished, mobile-first 3v3 hero arena prototype for web deployment. It includes:

- An original IP vision, world tone, style guide, and ranked presentation.
- A launch roster of 9 original heroes with bespoke identities and readable silhouettes.
- Three ranked-ready objective modes: Zone Control, Crystal Relay, and Knockout Protocol.
- A playable in-browser top-down scrim with desktop + touch controls and role-aware bots.
- Responsive command deck screens for roster, ranked systems, design guide, and gameplay preview.

## Run locally

Because this repository is dependency-free, you can open `index.html` directly in a browser, or serve it with a static server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Controls

- **Desktop:** `WASD` move, mouse aim, hold left mouse to fire, `Space` tactical, `Q` ultimate.
- **Mobile:** left-thumb movement stick, right-thumb aim/fire stick, tap for tactical, double tap for ultimate.
